🔐 Alarm Systeem Remote Debug - Ngrok

🚀 Ngrok Setup Instructies

Op je Raspberry Pi:

  1. Installeer Ngrok:
    wget https://bin.equinox.io/c/bNyj1mQuLUz/ngrok-v3-linux-arm.tgz
    tar -xvzf ngrok-v3-linux-arm.tgz
    sudo mv ngrok /usr/local/bin/
  2. Maak account op ngrok.com
  3. Kopieer Auth Token van je dashboard
  4. Configureer Ngrok:
    ngrok config add-authtoken JOUW_TOKEN_HIER
  5. Start Ngrok tunnel:
    ngrok http 5000
  6. Kopieer het HTTPS adres dat ngrok toont
⚠️ Belangrijk: Zorg dat je alarm systeem draait (python3 app.py) voordat je ngrok start!
